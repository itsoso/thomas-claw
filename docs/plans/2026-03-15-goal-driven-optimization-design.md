# 目标导向优化设计 — 从广撒网到深耕转化

## 目标
让妹子愿意线下见面。

## 当前漏斗
```
56 发现 → 15 注意到(27%) → 4 私信(7%) → 1 深入(1.8%) → 0 线下(0%)
```

## 三个优化方向

### 1. 深耕少数人
- 从已有反馈的主播里筛 3-5 个高潜力目标
- 高潜力：主播读过我们弹幕 + 人少 + 爱聊天
- 每天回访，停留更长，弹幕有记忆连续性
- 文件：修改 scheduler.ts + auto.ts

### 2. 精准发现
- 优先 5-50 人在线的小直播间
- 优先新人主播
- 避开商业直播间
- 文件：修改 discover.ts + taste.ts

### 3. 私信自动化
- 启动时检查所有已私信主播的回复
- 对话阶段化：破冰→了解→提议加微信
- 文件：修改 messenger.ts + auto.ts
